Absolute links are links to a full URL, which can be anywhere on the internet.
They require the http:// or https:// at the beginning.
This link will take you to google.

Relative links point to local content in your project directory. You only need to specify the file name and, if present, the folder structure.
If you click on index skeleton, you will open the one inside folder1 of the website.

Anchor links scroll the current page to the location of specific content. The href attribute points to an id present somewhere on the page
(indicated with a hashtag). No scrolling will happen if the browser window is too tall to allow it.
Click on Bread anchor

Email links, also known as mailto links open a new email draft, addressed to the specified recipient, in the default email app.
The syntax of the href attribute value is mailto: followed by the recipient email address with no spaces anywhere in the href.

The title attribute contains alternate text describing the link. Hover on Rainbow is in the sky

The target attribute. It forces a link to open in a new tab. The only useful value for target is _blank.
